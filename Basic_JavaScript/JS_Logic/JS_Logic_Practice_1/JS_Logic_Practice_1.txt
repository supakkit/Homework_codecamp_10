1) alert( null || 2 || undefined );
    Ans. 2  เนื่องจากมีตัวดำเนินการทางตรรกะแค่ OR(||) จึงเปรียบเทียบจากซ้ายไปขวา
            null เป็น false จึงเปรียบเทียบตัวถัดไปเมื่อเจอ 2 ซึ่งเป็น true
            จึง return ค่า 2 ออกมาพร้อมจบการเปรียบเทียบทันที

2) alert( alert(1) || 2 || alert(3) );
    Ans. 1 and 2    เนื่องจากมีตัวดำเนินการทางตรรกะแค่ OR(||) จึงเปรียบเทียบจากซ้ายไปขวา
                    เมื่อเจอ alert(1) จึงแสดงผล 1 ออกมาก่อน แต่ alert() เป็น function
                    ที่ไม่มีการ return ค่ากลับหรือก็คือ undefined จึงมีค่าเป็น false
                    จากนั้นเปรียบเทียบตัวถัดไปพบว่าเป็น 2 ซึ่งมีค่าเป็น true
                    จึง return พร้อมจบการเปรียบเทียบทันที

3) alert( 1 && null && 2 );
    Ans. null   เนื่องจากมีตัวดำเนินการทางตรรกะแค่ AND(&&) จึงเปรียบเทียบจากซ้ายไปขวา
                1 มีค่าเป็น true จึงเปรียบเทียบตัวถัดไปเมื่อเจอ null ซึ่งมีค่าเป็น false
                จึง return ค่า null พร้อมจบการเปรียบเทียบทันที

4) alert( alert(1) && alert(2) );
    Ans. 1  เนื่องจากมีตัวดำเนินการทางตรรกะแค่ AND(&&) จึงเปรียบเทียบจากซ้ายไปขวา
            เมื่อเจอ alert(1) จึงแสดงผล 1 ออกมาก่อน แต่ alert() เป็น function
            ที่ไม่มีการ return ค่ากลับหรือก็คือ undefined จึงมีค่าเป็น false
            ทำให้เงื่อนไขทั้งหมดเป็น false จึงจบการเปรียบเทียบทันที

5) alert( null || 2 && 3 || 4 );
    Ans. 3  เนื่องจากมีตัวดำเนินการทางตรรกะ AND(&&) และ OR(||) จึงเปรียบเทียบ AND(&&) ก่อน
            2 เป็น true จึงเปรียบเทียบตัวถัดไปคือ 3
            3 เป็น true ซึ่งเป็นการจบการดำเนินการทางตรรกะ AND แล้ว
            จากนั้นเป็นการดำเนินการทางตรรกะ OR ซึ่ง 3 เป็น true อยู่แล้ว
            จึง return ค่า 3 ออกมาพร้อมจบการเปรียบเทียบทันที
